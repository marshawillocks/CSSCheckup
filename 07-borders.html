Borders

The border property allows you to specify the style, width, and color of an element's borders.

Border style

The border-style property can take up to four values (top border, right border, bottom border, and left border). The following border-style values are allowed:

  • dotted:dotted border
  • dashed:dashed border
  • solid:solid border
  • double:double border
  • groove:3D grooved border, depends on border-color value
  • ridge3D ridge border, depends on border-color value
  • inset3D inset border, depends on border-color value
  • noneno border
  • hiddenhidden border
  • mixedyou can set a diffreent border style for the top, bottom, right, and left

Border Width

Very Important: border-width will not work unless you have a border-style set

border-width can be set to a specific size (px, em,etc) or you can use one of the following pres-set settings: thin, medium, thick. The border-widthproperty can also ake up to four values (top, right, bottom, left)
