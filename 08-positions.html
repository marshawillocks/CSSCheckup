Position

The position property specifies the type of positioning method used for an element. The four positions values are:

  • fixedThis will pin something to a spot, and it will stay as you scroll. (ex. nav bar) Fixed position takes elements out of a normal document flow
  • static:that is the default setting, it not affected by top, bottom, left, and right properties. It is always positioned according to the normal flow of the page
  • relative:does not take the element our of the normal document flow. still apperas within it's space/div/container,but also allows it to be offset a bit from the top, bottom, right, left. Top ight, bottom, and left properties will move it away from it's normal position. other content will not be adjusted to fit any gap left by the element/
  • absolute:takes the elekment out of the normal document flow. If a parent element poisiton is set to relative, and the child element is set to absolute, the child element's new (0,0) coordinate will be the upper left hand corner of the parent element instead of the whold page.
  • absolutetakes the element out of the normal document flow. if a parent element position is set to relative, and the child element set to absolute, the child element new (0,0) coordinate will be the upper left hand corner of the parent element instead of the whold page.
